# CS Hardware - Transistors

This lesson introduces transistors and digital logic. Participants will build transistor-controlled circuits, explore how small electrical signals control larger ones, and learn how transistors form the foundation of logic gates and modern computer hardware.

[Slides](https://dpi-tta-slides.github.io/cs-hardware-2-transistors/)

## Learning Objectives

By the end of this lesson, participants will be able to:

- Explain what a transistor does
- Identify transistor pins:
  - Base
  - Collector
  - Emitter
- Build a transistor-controlled LED circuit
- Explain how small current can control larger current
- Use transistors as electronic switches
- Understand binary ON/OFF states
- Build simple logic gate circuits
- Explore Boolean logic:
  - NOT
  - AND
  - OR
  - NAND
- Use buttons as logical inputs
- Relate logic gates to CPUs and modern computers
